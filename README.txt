Files:

- firebase-config.js
- manifest.webmanifest
- service-worker.js
- icon.svg

How to turn on shared updates:
1. Create a Firebase project.
2. Turn on Realtime Database.
3. Copy your Web App config from Firebase.
4. Paste that config into firebase-config.js.
5. Commit the updated files to GitHub.
6. Vercel will redeploy automatically.

Important:
- Everyone using the same deployed site will share the same rows, units, and checkmarks.
- The selected active unit stays personal to each device.
- Default admin PIN is 1234 unless you change it.
